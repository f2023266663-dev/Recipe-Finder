# 🍽️ RecipeRun - Find Your Next Meal

A full-stack recipe finder application where users can search for recipes, save favorites, and create their own recipes.

## 🚀 Live Demo

- **Frontend:** https://recipe-finder-indol-seven.vercel.app
- **Backend API:** https://recipe-finder-production-17b6.up.railway.app/api/test

---

## ✨ Features

- 🔍 Search thousands of recipes from TheMealDB API
- ❤️ Save favorite recipes (per user)
- 📝 Create and share your own recipes
- 🌓 Dark/Light mode toggle
- 🔒 User authentication (Login/Signup)
- 👤 Personal recipe management
- 🌍 Public/Private recipe visibility
- 📱 Fully responsive design

---

## 🛠️ Tech Stack

### Frontend
- **React.js** - UI Library
- **React Hooks** - State management
- **CSS** - Custom styling with dark mode support
- **Font Awesome** - Icons
- **TheMealDB API** - Recipe data
- **Vercel** - Deployment

### Backend
- **Node.js** - Runtime
- **Express.js** - Web framework
- **MongoDB Atlas** - Database
- **Mongoose** - ODM
- **JWT** - Authentication
- **bcryptjs** - Password hashing
- **Railway** - Deployment
